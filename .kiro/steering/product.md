# Product Overview

AIを使用したWiki生成プロジェクトです。Cursorエージェント用コマンドを通じて、指定されたディレクトリを分析し、Wiki構造を決定するためのトピック分析とWiki生成プロンプトを生成する機能を提供します。

## Core Capabilities

1. **Wikiトピック分析**: 指定ディレクトリの内容を分析し、適切なWiki構造を提案するプロンプトを生成
2. **Wiki生成プロンプト生成**: トピック分析結果から包括的なWiki生成プロンプトを生成
3. **Spec-Driven Development**: Kiro-styleの仕様駆動開発プロセスを実装し、段階的な開発を支援
4. **エージェントコマンド管理**: Cursorエージェント用のコマンド定義をMarkdownベースで管理

## Target Use Cases

- GitHubリポジトリやプロジェクトディレクトリから自動的にWiki構造を生成したい開発者
- プロジェクトの包括的なドキュメントを効率的に作成したいチーム
- Spec-Driven Developmentの手法で段階的に機能を開発したい開発者

## Value Proposition

- **自動化されたWiki構造生成**: ディレクトリ構造とREADMEを分析して、適切なWiki構造を自動提案
- **多言語対応**: 日本語を主要言語として、多様な言語でのWiki生成に対応
- **構造化された開発プロセス**: Requirements → Design → Tasks → Implementation の3段階承認ワークフローで、品質の高い開発を実現
- **エージェント統合**: Cursorエージェントとシームレスに統合され、コマンドベースで操作可能
