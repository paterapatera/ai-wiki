# AI Wiki

AIを使用したWiki生成プロジェクトです。

## 概要

このプロジェクトは、エージェント用コマンドを使用して、指定ディレクトリのWikiトピック分析とWiki生成プロンプトを生成する機能を提供します。

## 主な機能

- **Wikiトピック分析**: 指定ディレクトリのWikiトピックを分析するプロンプトを生成
- **Wiki生成プロンプト生成**: トピック分析結果からWiki生成プロンプトを生成
- **Wiki構造のJSON出力**: 指定ディレクトリを分析してWiki構造をJSON形式で出力

## 環境要件

### `/ai-wiki/init` コマンドの実行環境

- **Docker**: JSONファイルの検証に使用します（必須）
- **Python/Node.js**: ホスト環境にはインストール不要です（Dockerコンテナ内で使用されます）

### 動作確認済み環境

- Dockerがインストールされ、実行可能な状態であること
- Cursor IDEのエージェント機能が利用可能であること
