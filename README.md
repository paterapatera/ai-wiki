# AI Wiki

AIを使用したWiki生成プロジェクトです。

## 概要

このプロジェクトは、エージェント用コマンドを使用して、指定ディレクトリのWikiトピック分析とWiki生成プロンプトを生成する機能を提供します。

## 主な機能

- **Wikiトピック分析**: 指定ディレクトリのWikiトピックを分析するプロンプトを生成
- **Wiki生成プロンプト生成**: トピック分析結果からWiki生成プロンプトを生成

このプロジェクトは、[deepwiki-open](https://github.com/AsyncFuncAI/deepwiki-open)の一部ソースコードを使用しています。
